요약
	1. SPCleaner_Search 콘솔파일을 실행 -> 사용자가 설치한 보안프로그램 및 은행목록을 생성
	2. eliminator_3 GUI 파일 실행	-> 사용자에게 현재 있는은행을 보여주고, 삭제하지않을 은행목록을 받음
	3. 비교 콘솔파일 실행 -> 받은 은행목록으로 삭제하지 않을 파일을 제외한, 파일을 기록한 텍스트파일 생성
	4. eliminator_5 GUI 파일 실행 -> 삭제할 프로그램 목록을 출력하고, 삭제하지 않을 프로그램 목록을 받음
	5. 삭제 콘솔파일 실행 -> 삭제


찾기
	-> 파일을 찾아 현재존재하는 파일 + 은행을 파일에 0, 1형태로 저장
	-> 처음 21개는 파일, 이후 17개는 은행을 나타냄 
	-> classifiedBK 파일에 현재 존재하는 은행을 번호 : 은행명으로 기록


eliminator_3 실행
	-> classifiedBK 파일을 참조하여 현재 있는 은행명을 출력
	-> 삭제하면 안되는 기관 목록을 받아서 choose 파일에 저장
		입력이 변할때마다 저장하여, 입력받은 대로 저장한다.
	-> 이후 비교 실행.


비교
	-> record 파일에서 현재 존재하는 보안프로그램 목록 + 은행별 가지는 프로그램 목록을 받음
	-> choose 파일에서 사용자가 고른 은행을 받음
	-> deinstall 파일에 현재 존재하는 보안프로그램에서 사용자가 고른 은행이 가지는 프로그램을 제외한 목록을 재생성
